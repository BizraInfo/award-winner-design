NODE0 · LOCAL-FIRST CLI

INSTALL DEMA

Dema is the human-facing CLI of BIZRA Node0 — a local-first mission cockpit. No daemon, no background process, no network calls by default, no account. Everything it does leaves a readable receipt on your own disk.

Requirements

  • Node.js ≥ 20 (nodejs.org)
  • Git
  • Linux, macOS, or Windows (WSL recommended)

Install & run

# zero dependencies — clone and run
git clone https://github.com/BizraInfo/Dema.git
cd Dema
node bin/dema

That's the whole install. The bare command shows the banner, current setup state, and the next safe step. Try node bin/dema welcome for first-run orientation, or node bin/dema --help for the full surface.

What you get today

MEASUREDDeterministic local kernels: mission previews, receipt read/list, consent-gated sandbox actions, proof-chain verification — each backed by tests in the repo.

PREVIEWFederation, token economy, and Proof-of-Impact rewards are design surfaces only — previewed and labeled in the CLI, not live. Nothing mints, nothing trades, nothing phones home.
